# Payment


Простой и удобный сервис на Django для ввода показаний счётчиков и автоматического расчёта итоговых коммунальных платежей.

## 🚀 Возможности

- Ввод текущих и предыдущих показаний:
  - холодная вода
  - горячая вода
  - водоотведение
  - электричество
  - интернет
- Автоматический расчёт стоимости по тарифам
- Сохранение истории всех расчётов
- Удобный интерфейс на Bootstrap с кастомной цветовой темой
- Защита от повторной отправки формы (паттерн POST‑Redirect‑GET)


## 🏗️ Технологии

- **Python 3**
- **Django**
- **PostgreSQL**
- **Bootstrap 5**
